## 주요 기능🐶
### 🐾반려인을 위한 커뮤니티
- 커뮤니티에서 자유롭게 글과 댓글로 소통 가능
### 🐾반려동물과 동반 가능한 장소(숙소, 식당, 카페 등) 정보 제공
- 메인 페이지에서 인기 높은 6개 장소 정보 제공
- 원하는 장소 예약 및 결제 가능
- 예약한 장소에 대한 후기 작성 가능
- 다른 사용자들이 만든 후기 조회 가능
