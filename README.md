# 해당 서비스 실행 시!
## 1. http://localhost:8088/petplanet 으로 접속
## 2. 카카오 회원가입을 통해 서비스에 가입한다.
### 2-1. 만약 회원가입하지 않고 서비스를 실행하고 싶다면
   http://localhost:8088/main/1 로 이동하여 서비스에 바로 접속할 수 있다.
